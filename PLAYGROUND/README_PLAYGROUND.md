# 🎡 Bem-vindo ao seu PLAYGROUND LOCAL

Este é o ambiente onde você tem **liberdade total** para testar o sistema Gravity sem medo de quebrar nada real!

### 🏗️ O que temos aqui:
1.  **Banco de Dados Isolado**: Você tem dois bancos locais (`configurador_test` e `tenant_test`). Tudo o que você fizer aqui **NÃO afeta** o sistema real nem o faturamento no Stripe.
2.  **Ambientes (Env)**: Já configuramos os arquivos `.env.test`. O sistema sabe automaticamente que, quando você rodar um comando de teste, ele deve usar esses arquivos.

### 🕹️ Como usar (Próximos Passos):
1.  **Novo Terminal**: Sempre que quiser rodar o Playground, abra um novo terminal.
2.  **Comando Mágico**: Em breve, vou te passar um comando único (como `npm run play`) que vai ligar o seu Front, o seu Back e conectar no seu Banco de Teste tudo de uma vez.

### 🛡️ Regras do Playground:
- **Não use senhas reais**: Use senhas simples para seus usuários de teste.
- **Limpeza**: A qualquer momento, eu posso rodar um comando que "zera" o banco e deixa ele limpinho com dados padrão para você recomeçar.
