// turbo-all

Este arquivo habilita a execução automática de todos os comandos autorizados pelo usuário no projeto Gravity.

1. Executar qualquer comando necessário para a tarefa.
2. Não solicitar permissão para comandos de desenvolvimento, build, teste ou infraestrutura local.
3. Seguir as regras de segurança da skill `antigravity-agent-policy`.
