## Como ler

- **Local**: produto/servico que possui o model.
- **Nome DDD**: nome PascalCase do model Prisma (com `@@map("snake_case")`).
- **Tipo**: agregado, entidade, value-object, etc.
- **Tem tenant_id?**: se possui `id_organizacao` (REGRA universal de isolamento).
- **PK**: chave primaria.
- **Relacoes / Indices**: relacionamentos e indices declarados.
- **Soft delete / Auditoria**: marcadores de governanca.
- **Arquivo fragment**: caminho do `fragment.prisma`.
- **Descricao**: o que o model representa.
- **Alias historico**: nome antes do rename DDD.

Convencoes:
- Apenas valores DDD-finais. Nao mostra estado pre-rename.
- Onde aplicavel, coluna "Alias historico" mostra nome legado (util para grep e git log --follow).
- Linhas marcadas `—` na planilha estao no apendice (nao sao acionaveis).
