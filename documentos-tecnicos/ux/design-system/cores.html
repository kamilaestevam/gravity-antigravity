4 — Os 3 Pontos de Contato

A cor de acento de cada produto aparece em exatamente 3 lugares. Em todo o resto, a UI usa neutros Solid Slate.

Ponto de Contato Localização Implementação CSS Intensidade
① Chip do produto Topbar — lado esquerdo, ao lado do logo background: linear-gradient(135deg, {color}18, {color}08) Muito sutil — 8–10% opacidade
② Indicador da sidebar Menu lateral — ponto colorido no item ativo background: {color} — dot 6×6px Direto — 100% cor
③ Item ativo na navegação Menu lateral — fundo do item selecionado background: {color}12 — texto: {color} Sutil — 7% opacidade

5 — Regras Obrigatórias

Toda implementação nova deve respeitar estas regras.

  • 1Cor de acento via prop, nunca hardcoded no componente. O componente TelaProdutoGlobal recebe productColor como prop. O valor vem de getProdutoMeta(productId).color.
  • 2Fonte única de verdade: nucleo-global/Logo/produtos/src/produtos.tsx — PRODUTO_META. Nunca duplicar cor em outro arquivo.
  • 3Máximo 3 pontos de contato por tela. Cores de produto nunca em: fundos de página, títulos, botões primários, badges de status, bordas de tabela.
  • 4Sistema de zonas = indigo unificado. HUB, Core e Admin usam #818cf8. Configurador usa #f472b6 por ser um produto de workspace (diferente dos produtos COMEX mas diferente das zonas neutras também).
  • 5KPI cards com accent top-border: border-top: 2px solid {color} — o único lugar onde a cor aparece em cards de dados.
  • 6Botões primários sempre indigo (var(--accent)) — nunca na cor do produto. Botões usam a cor da plataforma, não do produto.
  • 7Light theme: as mesmas cores são mantidas. No light theme, os chips ficam mais saturados pois o fundo é branco — compensar com opacidade menor (06–08%).

6 — Tokens de Cor (Referência)

Cores do PRODUTO_META — única fonte de verdade. Importar via @nucleo/logo-produtos.

// nucleo-global/Logo/produtos/src/produtos.tsx
// getProdutoMeta(productId).color — Fonte de verdade

PRODUTO_META = {
  'simula-custo':    color: '#34d399',  // Emerald 400
  'pedido':    color: '#f59e0b',  // Amber 400
  'bid-cambio':    color: '#06b6d4',  // Cyan 500
  'bid-frete':    color: '#60a5fa',  // Orange 500
  'lpco':    color: '#f43f5e',  // Orange 400
  'nf-importacao':    color: '#c084fc',  // Purple 400
  'processo':    color: '#facc15',  // Yellow 400
  'financeiro-comex':   color: '#f472b6',  // Pink 400 ⚠ ver nota colisão
}

// Zonas do Sistema (não estão no PRODUTO_META)
GRAVITY_INDIGO = '#818cf8' // HUB, Core, Admin — Indigo 400
CONFIGURADOR = '#f472b6' // Configurador — Pink 400
