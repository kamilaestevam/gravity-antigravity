# Mapeamento de Campos de Dados (Mirroring Map)

Este documento registra a equivalência de nomenclatura entre as telas administrativas globais (Admin Panel) e as telas de configuração de organização (Configurador).

> [!IMPORTANT]
> Manter estes nomes idênticos é fundamental para a integridade dos dados e transparência para o usuário final.

| Admin Panel (Visão Geral) | Configurador (Organização) | Definição / Campo Banco |
|---|---|---|
| Nome da Empresa | Nome da Empresa | Identificador social/fantasia principal |
| CNPJ | CNPJ | Registro fiscal nacional (BR) |
| Segmento | Segmento | Setor de atuação da organização |
| Site | Site | URL oficial da empresa |
| Subdominio | Subdominio | Prefixo de rede que individualiza a instância |
