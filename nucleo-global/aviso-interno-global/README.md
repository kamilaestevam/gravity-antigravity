# aviso-interno-global

Este é o componente visual responsável por renderizar a área de Notificações Internas (Sininho) da plataforma. 
Ele não contém estado interno de busca (`fetch`) nem regras de negócio, recebendo tudo via `props`.

Pode ser usado no Shell ou em serviços de tenant (como o `ServicoTemporarioNotificacoes`).
